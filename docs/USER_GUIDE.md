# Guía de Usuario - CivilProtect

## Acceso
1. Navega a la URL de la aplicación.
2. Regístrate o Inicia Sesión con tus credenciales.

## Creación de Análisis
1. **Pestaña "Nuevo Análisis"**:
   - Completa Ubicación y Características del Inmueble (m², niveles, uso).
   - Responde el cuestionario de Infraestructura (Gas, Transformadores, etc.).
2. **Resultados**:
   - El sistema procesará los datos mediante cálculo matemático e Inteligencia Artificial.
   - Visualiza el cumplimiento normativo en pantalla.
   - Descarga el reporte PDF preliminar con justificación jurídica.
   - Firma digitalmente para validar el documento final.

## Historial
- La pestaña "Historial" muestra tus análisis previos.
- Puedes ver detalles, volver a descargar el PDF o eliminarlos.
- Los análisis se almacenan de forma segura y privada.

## Soporte
- Si encuentras errores, el sistema los reportará automáticamente a nuestro equipo técnico.
- Para ayuda adicional, contacta a soporte@civilprotect.local.
